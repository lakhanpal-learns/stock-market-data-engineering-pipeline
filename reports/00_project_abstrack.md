# Project Abstract

## Title

# Stock Market Data Engineering Pipeline

---

## Abstract

The Stock Market Data Engineering Pipeline is an end-to-end data engineering project developed to demonstrate the process of collecting, processing, transforming, and visualizing financial market data.

The project retrieves historical stock market data from Yahoo Finance using the yfinance Python library and processes it through a modular data pipeline. The pipeline performs data validation, cleaning, transformation, feature engineering, and technical indicator generation before delivering the processed data to an interactive Streamlit dashboard.

The dashboard enables users to explore historical stock performance through interactive visualizations, key performance metrics, and downloadable datasets. Features such as Moving Averages (MA20 and MA50), Relative Strength Index (RSI), daily returns, volatility, and statistical trend estimation provide additional analytical insights.

The project follows fundamental data engineering principles including modular architecture, reusable pipeline design, structured data processing, and separation of concerns. Rather than focusing on machine learning-based forecasting, the project emphasizes building a reliable analytics-ready data pipeline capable of supporting future analytical and machine learning applications.

This project demonstrates practical applications of Python, Pandas, NumPy, Plotly, Streamlit, and Yahoo Finance while showcasing the complete lifecycle of financial data preparation from raw data acquisition to interactive visualization.
